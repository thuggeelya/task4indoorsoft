# Admin Panel
### Two entities:
- Employee
- Project
### Two Screens:
- Employees info editing
- Projects info and participants editing

## Deploy
Run CUBA Application and enter the credentials 'admin' and 'admin' as the password.
The screens are located on the 'Menu' tab.

## Stack
- CUBA 7.2.20
- HSQL DB 2.4.1
- Java 1.8.0
